# ai-smart-product-scanner
AI Smart Product Scanner is a computer vision–based assistive system that extracts key product information such as expiry date, MRP, and common allergen warnings using deep learning–based OCR and NLP techniques.
# AI Smart Product Scanner

## Overview
An AI-based product scanner that extracts:
- Expiry date
- MRP
- Common allergen warnings

Uses OCR (EasyOCR) and NLP-based pattern detection.

## Features
- Image upload
- Expiry date detection
- MRP extraction
- Allergy keyword detection
- Voice feedback

## How to Run

1. Clone repository
2. Install requirements:
   pip install -r requirements.txt
3. Run:
   streamlit run app.py
